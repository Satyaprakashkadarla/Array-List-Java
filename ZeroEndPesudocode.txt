function moveZerosToEnd(arr):
    left = 0
    right = 0

    while right < length(arr):
        if arr[right] != 0:
            swap arr[left] and arr[right]
            increment left
        increment right

    return arr


//This pseudocode represents the same logic as the Python implementation provided earlier. It uses two pointers, left and right, to traverse the array and swaps non-zero elements with elements at the left pointer while incrementing both pointers.
